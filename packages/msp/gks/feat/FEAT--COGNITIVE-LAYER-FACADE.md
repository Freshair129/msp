# FEAT — createCognitiveLayer

## User-facing behaviour

```ts
import { createCognitiveLayer } from 'msp'

const layer = await createCognitiveLayer({
  root: process.cwd(),
  slm: { tier: 'T1' },              // default — local Ollama + qwen2.5-coder
})

// 4-layer hybrid recall (FRAMEWORK_MASTER_SPEC §13)
const hits = await layer.recall('how did we decide rate limiting?')

// Episodic results are stamped audit_only (§7.5)
hits.hits.filter((h) => h.audit_only)

// Write path
await layer.remember('Cortex handles planning.', { tags: ['cortex'] })

// Codegen with scale-level gate (§7.7.2)
await layer.runTask('./.brain/tasks/FEAT--X/T1.task.yaml', { scale: 'L2' })

// SSOT conflict resolver (§14.1)
const winner = layer.resolveSSOT([
  { id: 'PROTO--X', type: 'proto', source: 'atom' },
  { id: 'ADR--X',   type: 'adr',   source: 'atom' },
])
// → { id: 'PROTO--X', ... }

// Hotfix escape hatch (§6.4)
await layer.hotfix.open({ sha: 'abc1234', reason: 'urgent rollback' })

// Spawn the existing 19-tool MCP surface for the consuming agent
const server = layer.mcpServer()
```

## Acceptance criteria

- [ ] `createCognitiveLayer({ root })` resolves without `slm` / `graphBackend` / `obsidian` opts.
- [ ] `layer.remember()` returns `{ id: string }`.
- [ ] `layer.recall()` returns a `CognitiveRecallResult` whose hits carry `audit_only: true` for episodic source.
- [ ] `layer.runTask()` with `scale: 'L2'` and a blueprint missing CONCEPT / ADR / FEAT throws `ScaleLevelGateError`.
- [ ] `layer.mcpServer()` returns a `McpServer` instance.
- [ ] Default `slm.tier: 'T1'` resolves to provider `'ollama'` (validated via `tier-routing.test.ts`).
